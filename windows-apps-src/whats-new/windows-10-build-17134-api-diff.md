# <a name="new-apis-in-windows-10-build-17134"></a>Nouvelles API de Windows 10, build 17134

Pour des informations sur les API ajoutées dans la version publique précédente, voir [Nouvelles API de Windows 10 Fall Creators Update](windows-10-build-16299-api-diff.md).
