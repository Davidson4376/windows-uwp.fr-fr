# <a name="new-apis-in-the-windows-10-sdk-preview-build-16190"></a>Nouvelles API dans le Kit de développement logiciel Windows10 version préliminaire16190

Des espaces de noms d’API nouveaux et modifiés ont été mis à disposition des [Windows Insiders](https://insider.windows.com/) dans le Kit de développement logiciel Windows10 version préliminaire16190. Cette version accompagne la [conférence des développeurs MicrosoftBuild2017](https://developer.microsoft.com/windows/projects/events/build/2017?ocid=wdgbld17_intreferral_devcenterhp_null_null_devcenter_hppost&utm_campaign=wdgbld17&utm_medium=internalreferral&utm_source=devcenterhp&utm_content=devcenter_hppost).

Vous trouverez ci-dessous une liste complète des ressources de documentation de versions préliminaires publiées sur les espaces de noms ajoutés ou modifiés depuis la dernière version publique de Windows10, [version 1703](windows-10-version-1703-api-diff.md). Notez que la documentation sur les versions préliminaires peut être incomplète et faire l’objet de modifications.
