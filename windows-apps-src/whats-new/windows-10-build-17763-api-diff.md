# <a name="new-apis-in-windows-10-build-17763"></a>Nouvelles API dans Windows 10 build 17763

Espaces de noms API nouvelles et mises à jour ont été mises à disposition pour les développeurs dans Windows 10 build 17763 (également appelé octobre 2018 version 1809 ou mise à jour). Vous trouverez ci-dessous une liste complète de ressources concernant la documentation publiée pour les espaces de noms ajoutés ou modifiés dans cette version.

Pour plus d’informations sur les API ajoutées dans la version publique précédente, consultez [nouvelles API dans la mise à jour de Windows 10 avril](windows-10-build-17134-api-diff.md).
