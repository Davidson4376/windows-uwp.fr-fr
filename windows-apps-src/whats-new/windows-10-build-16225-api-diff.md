# <a name="new-apis-in-the-windows-10-sdk-preview-build-16225"></a>Nouvelles API dans le SDKWindows10 version d'évaluation16225

Des espaces de noms d’API nouveaux et modifiés ont été mis à disposition des [Windows Insiders](https://insider.windows.com/) dans le SDKWindows10 version d'évaluation16225.

Vous trouverez ci-dessous une liste complète des ressources de documentation de versions d'évaluation publiées pour les espaces de noms ajoutés ou modifiés depuis la dernière version publique de Windows10, [version1703](windows-10-version-1703-api-diff.md). **Veuillez noter que la documentation de versions d'évaluation peut être incomplète et sujette à modification, et que ces API peuvent être renommées ou supprimées dans les builds suivantes.**
