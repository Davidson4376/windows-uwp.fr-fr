# <a name="usersxuidxuid"></a>/users/xuid({xuid})
Accéder à la présence d’un autre utilisateur ou un client. Le domaine pour ces URI est `userpresence.xboxlive.com`.
 
  * [Paramètres d’URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Paramètres d’URI
 
| Paramètre| Type| Description| 
| --- | --- | --- | 
| xuid| entier non signé 64 bits| Xbox utilisateur ID (XUID) de l’utilisateur cible.| 
  
<a id="ID4EUB"></a>

 
## <a name="valid-methods"></a>Méthodes valides

[GET (/users/xuid({xuid}))](uri-usersxuidget.md)

&nbsp;&nbsp;Détecter la présence d’un autre utilisateur ou un client.
