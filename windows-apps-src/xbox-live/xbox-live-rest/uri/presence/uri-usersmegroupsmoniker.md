# <a name="usersmegroupsmoniker"></a>/users/me/groups/{moniker}
Accède à la PresenceRecord pour mon groupe. Le domaine pour ces URI est `userpresence.xboxlive.com`.
 
  * [Paramètres d’URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Paramètres d’URI
 
| Paramètre| Type| Description| 
| --- | --- | --- | 
| moniker| chaîne| Chaîne définissant le groupe d’utilisateurs. Le moniker du acceptées uniquement à l’heure actuelle est « People », avec un « P » majuscule.| 
  
<a id="ID4ERB"></a>

 
## <a name="valid-methods"></a>Méthodes valides

[OBTENIR (/ users/me/groupes / {nom})](uri-usersmegroupsmonikerget.md)

&nbsp;&nbsp;Obtient le PresenceRecord pour mon groupe.
