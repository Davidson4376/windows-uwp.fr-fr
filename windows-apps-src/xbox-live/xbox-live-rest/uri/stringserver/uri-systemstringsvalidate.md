# <a name="systemstringsvalidate"></a>/system/strings/validate
Accède à un tableau de chaînes pour la validation. Le domaine pour ces URI est `client-strings.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Méthodes valides

[POST (/system/strings/validate)](uri-systemstringsvalidatepost.md)

&nbsp;&nbsp;Accepte un tableau de chaînes pour la validation et retourne un tableau de résultats de taille égale.
