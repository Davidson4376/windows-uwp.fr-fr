# <a name="usersxuidxuidlistspinslistnamecontainsitems"></a>/users/xuid(xuid)/lists/PINS/{listname}/ContainsItems
Détermine si un ensemble d’éléments (spécifié par itemId) sont contenus dans une liste sans récupérer la liste entière. Le domaine pour ces URI est `eplists.xboxlive.com`.
 
  * [Paramètres d’URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Paramètres d’URI 
 
| Paramètre| Type| Description| 
| --- | --- | --- | 
| XUID| chaîne| XUID de l’utilisateur.| 
| nom de la liste| chaîne| Nom de la liste à manipuler.| 
  
<a id="ID4E5B"></a>

 
## <a name="valid-methods"></a>Méthodes valides

[POST](uri-usersxuidlistspinslistnamecontainsitemspost.md)

&nbsp;&nbsp;Détermine si une liste contient un ensemble d’éléments (spécifié par itemId) sans récupérer la liste entière.
