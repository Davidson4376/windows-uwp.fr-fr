# <a name="usersxuidxuidlistspinslistnameremoveitems"></a>/users/xuid(xuid)/lists/PINS/{listname}/RemoveItems
Supprime les éléments d’une liste. Le domaine pour ces URI est `eplists.xboxlive.com`.
 
  * [Paramètres d’URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Paramètres d’URI 
 
| Paramètre| Type| Description| 
| --- | --- | --- | 
| XUID| chaîne| XUID de l’utilisateur.| 
| nom de la liste| chaîne| Nom de la liste à manipuler.| 
  
<a id="ID4E5B"></a>

 
## <a name="valid-methods"></a>Méthodes valides

[SUPPRIMER](uri-usersxuidlistspinslistnameremoveitemsdelete.md)

&nbsp;&nbsp;Supprime les éléments d’une liste par index.

[POST](uri-usersxuidlistspinslistnameremoveitemspost.md)

&nbsp;&nbsp;Supprime les éléments d’une liste par itemId.
