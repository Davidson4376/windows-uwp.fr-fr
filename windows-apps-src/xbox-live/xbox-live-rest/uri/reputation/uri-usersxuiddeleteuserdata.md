# <a name="usersxuidxuiddeleteuserdata"></a>/users/xuid({xuid})/deleteuserdata
Réinitialise entièrement les données de réputation d’un utilisateur de test. Fins de test uniquement. Le domaine pour ces URI est `reputation.xboxlive.com`. Cet URI est toujours appelé sur le port 10443.
 
  * [Paramètres d’URI](#ID4EV)
 
<a id="ID4EV"></a>

 
## <a name="uri-parameters"></a>Paramètres d’URI
 
| Paramètre| Type| Description| 
| --- | --- | --- | 
| xuid| entier non signé 64 bits| Xbox utilisateur ID (XUID) de l’utilisateur dont les données sont en cours de suppression.| 
  
<a id="ID4EYB"></a>
