# <a name="usersmeinventory"></a>/users/me/inventory
Accède à l’ensemble de l’inventaire actuellement associé à l’utilisateur spécifié. Le domaine pour ces URI est `inventory.xboxlive.com`.
  
<a id="ID4EV"></a>

 
## <a name="valid-methods"></a>Méthodes valides

[OBTENIR (/ utilisateurs/me/inventaire)](uri-inventoryget.md)

&nbsp;&nbsp;Fournit l’ensemble de l’inventaire actuellement associé à l’utilisateur fourni à l’appelant.
