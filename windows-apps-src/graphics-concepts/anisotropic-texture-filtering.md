# <a name="anisotropic-texture-filtering"></a>Filtrage de textures anisotropiques


*L'anisotropie* désigne la distorsion visible dans les texels d'un objet3D dont la surface est orientée à un certain angle par rapport au plan de l'écran. Un pixel d’une primitive anisotropique apparaît déformé lorsqu'il est mappé à des texels. Direct3D mesure l’anisotropie d’un pixel par l’allongement (autrement dit, la longueur divisée par la largeur) d’un pixel de l'écran inversé dans l’espace de texture.

Vous pouvez utiliser le filtrage de textures anisotropiques parallèlement à un filtrage de textures linéaires ou à un filtrage de textures mipmap afin d'améliorer les résultats de rendu.
